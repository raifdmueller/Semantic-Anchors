Vollständiger Name:: GoF Interpreter Pattern (Behavioral)

[discrete]
== *Absicht*:
Definiere für eine gegebene Sprache eine Repräsentation ihrer Grammatik zusammen mit einem Interpreter, der diese Repräsentation nutzt, um Sätze in der Sprache zu interpretieren.
