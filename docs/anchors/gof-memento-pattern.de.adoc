Vollständiger Name:: GoF Memento Pattern (Behavioral)

[discrete]
== *Absicht*:
Erfasse und externalisiere den internen Zustand eines Objekts, ohne die Kapselung zu verletzen, sodass das Objekt später in diesen Zustand zurückversetzt werden kann.
