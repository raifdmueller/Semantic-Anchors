Vollständiger Name:: GoF Proxy Pattern (Structural)

[discrete]
== *Absicht*:
Stelle einen Stellvertreter oder Platzhalter für ein anderes Objekt bereit, um den Zugriff darauf zu kontrollieren.

[discrete]
== *Wann zu verwenden*:
* Wenn du eine vielseitigere oder ausgefeiltere Referenz auf ein Objekt benötigst als einen einfachen Zeiger (Remote-, Virtual-, Protection- oder Smart-Reference-Proxy)

[discrete]
== *Prompt-Beispiel*:
"Implementiere einen Caching-Proxy nach GoF für den API-Client, der wiederholte Anfragen aus dem Cache beantwortet."
