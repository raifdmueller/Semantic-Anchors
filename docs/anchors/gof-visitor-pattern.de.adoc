Vollständiger Name:: GoF Visitor Pattern (Behavioral)

[discrete]
== *Absicht*:
Repräsentiere eine Operation, die auf den Elementen einer Objektstruktur ausgeführt werden soll. Visitor ermöglicht es, eine neue Operation zu definieren, ohne die Klassen der Elemente zu ändern, auf denen sie operiert.
